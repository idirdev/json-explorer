# JSON Explorer

Interactive JSON viewer and editor. Paste or type JSON, see a collapsible tree view, format/minify, and inspect values by type.

## Features

- Split view: editor with line numbers | collapsible tree viewer
- Type-colored values (string, number, boolean, null)
- Format, minify, sort keys, copy, download
- Error detection with line/column location

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).

## 🇫🇷 Documentation en français

### Description
JSON Explorer est un visualiseur et éditeur JSON interactif. Collez ou saisissez du JSON pour obtenir une vue arborescente repliable, avec coloration des types, formatage et minification en un clic. Les erreurs de syntaxe sont détectées avec indication de la ligne et de la colonne.

### Installation
```bash
npm install
npm run dev
```

Ouvrez [http://localhost:3000](http://localhost:3000) dans votre navigateur.

### Utilisation
L'interface est divisée en deux panneaux : un éditeur avec numéros de ligne à gauche, et une arborescence interactive à droite. Utilisez les boutons **Format**, **Minify**, **Sort Keys**, **Copy** et **Download** pour manipuler vos données JSON.
